查看运行时日志
====================

###有二种办法查看服务端运行日志:

1: 使用可视化的控制台工具[GUIConsole], 必须开启[messagelog]服务。

2: 查看log文件([KBE_BIN_PATH]/[logs]/xxx.log)。


-----------------------------------------------------------------------------------------------

###日志类型:

	DEBUG		: 调试日志
	WARRING		: 警告日志
	ERROR		: 错误日志
	INFO		: 常规日志
	S_DBG		: 脚本层调试日志
	S_ERR		: 脚本层错误日志
	S_INFO		: 脚本层常规日志
